# djv1-TPS 



## Contrôle 

* zqsd : Pour le déplacement
* Barre espace : Pour sauter
* x : équiper / déséquiper l'arme
* c : Roulade avant
* 1, 2 : Changer les armes
* Shit : Pour courir
* clic-droit pour viser
* clic-gauche pour tirer.
